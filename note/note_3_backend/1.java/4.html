第四章 类与对象

4.1 类

类是 Java 程序的基本要素，一个 Java 应用程序就是由若干类所构成的。类是 Java 语言中最重要的数据类型，类声明的变量被称作对象变量，简称对象。

编程风格（大驼峰）

类的名字要符合标识符规定。在给类命名时要遵守下列编程风格：

  • 如果类名使用拉丁字母，那么名字的首字母使用大写字母。
  • 类名最好容易识别、见名知意。
  • 当类名由多个单词复合而成时，每个单词的首字母应大写。

类体

定义类的目的是抽象出一类事物共有的属性和行为，并用一定的语法格式来描述所抽象出的属性和行为。

抽象的关键是抓住事物的两个方面——属性和行为，即数据以及在数据上所进行的操作，因此类体的内容由如下所述的两部分构成：

  • 变量的声明
  • 方法的定义

成员变量

类体中的内容可分为两部分：一部分是变量的声明；另一部分是方法的定义。声明变量部分所声明的变量被称为成员变量或域变量。

成员变量的类型

成员变量的类型可以是 Java 中的任何一种数据类型，包括：

  • 基本类型：整型、浮点型、字符型、逻辑类型。
  • 引用类型：数组、对象和接口。

成员变量的默认值和有效范围

在声明成员变量时如果没有指定初始值， Java 编译器会为其指定默认值：

变量类型 默认值
boolean false
byte 0
short 0
int 0
long 0
char \0
float 0.0
double 0.0
引用型变量（数组、对象和接口） null

成员变量在整个类的所有方法里都有效，其有效性与它在类体中出现的位置无关。

不提倡把成员变量的声明分散地写在方法之间，人们习惯先描述属性再描述行为。

编程风格（小驼峰）

  • 一行只声明一个变量。
  • 变量名字要符合标识符规定。
  • 变量名字首单词的首字母使用小写，从第 2 个单词开始首字母大写。
  • 变量名字要见面知意。
