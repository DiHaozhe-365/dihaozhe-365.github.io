数据库设计

版本信息

  • 数据库服务：MySQL 8.0.34
  • 数据库工具：Navicat 16

数据库

  • 数据库名称：db_flower_studio
  • 字符集：utf8mb4
  • 排序规则：utf8mb4_general_ci

数据表

  • 商家表：tb_store

    名 类型 长度 小数点 不是 null 虚拟 键 注释 其他设置

    phone varchar 256 ✅ 预留电话

  • 客户表：tb_client

    名 类型 长度 小数点 不是 null 虚拟 键 注释 其他设置
    id int ✅ 🔑 客户 ID auto_increment
    store_id int ✅ 商家 ID

    phone varchar 128 客户电话
    address varchar 1024 客户地址

    create_time datetime 创建时间（数据插入时间） default CURRENT_TIMESTAMP

    on update CURRENT_TIMESTAMP
    is_delete varchar 128 ✅ 是否删除 未删除/已删除
